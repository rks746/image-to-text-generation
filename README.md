Project: Image-to-Text Generation and Analysis Platform (GenAI)

Description:

A platform leveraging Generative AI to analyze images-such as diagrams, charts, or handwritten notes-and generate descriptive text or structured data outputs.

Developed modules for:

1. Extracting text from diagrams or handwritten notes using OCR. 
2. Generating natural language descriptions of images. 
3. Converting image data into structured tables or mind maps.
